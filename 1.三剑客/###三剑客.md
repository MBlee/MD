## Redux

- Store：数据仓库

- State：1个对象，Store所有数据都放到1个State
- Action：1个动作，触发数据改变的方法
- Dispatch：将动作触发成方法
- Reducer：1个函数，通过获取动作，改变数据，生成1个新State，从而改变页面

~~~js
import {createStore}
~~~

~~~js
// 创建Store
createStore((state={},action)=>{
    action.type...
    return {...state}
}）
// 获取数据
Store.getState()
// 修改数据
Store.dispatch({type:})
// 监听数据
Store.subscibe(()=>{render...})
~~~
