# 开始

> 快速启动：nodemon --watch main.js --exec npm run build 

~~~shell
# 克隆模板
git clone https://github.com/electron/electron-quick-start
# 脚手架
create-electron-app xxx_app
# 包
npm i electron -g || yarn add electron
# yarn构建
yarn create electron-app xxApp
yarn start
yarn make
~~~
