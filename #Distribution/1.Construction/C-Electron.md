## Forge

```shell
create-electron-app <app> #Dev
--tempalte=webpack-typescript(vite-teypescript) 
1. electron-forge start #Runner
2. electron-forge package #Package
3. electron-forge make #Maker
4. electron-forge publish (npm:@electron-forge/publisher-github) #Publisher

# Manual
npm: @electron-forge/cli
npm exec --package=@electron-forge/cli -c "electron-forge import"
```
