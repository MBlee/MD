## 🚀CLI

```shell
#💥💥💥npm: @nestjs/cli
nest new/n <app>
nest generate/g <schematic> <name>
- schematic: service/controller/module/resource
- --flat/no-spec
```

```shell
#🚗Start
nest start <name>
--debug <port>
--watch
--builder
```

```shell
#🚗Build
nest build <name>
--builder
```

```shell
#🚗Lib
nest add <lib>
nest info
```

```shell
#🚗Monorepo
# Workspace
nest-cli.json => root
nest generate app <app>
nest generate library <lib>
# Library
monorepo => libs/**
```
