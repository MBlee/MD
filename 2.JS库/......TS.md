## 数据类型

~~~js
// 基本类型（多类型|&）
- any, unknown
- void, never
// 对象类型
- object|{xx?|[xx:string]:any}
// 数组类型
- string[],Array<string>
- [string,string]
// 枚举类型
- enum xx{k=v,...}
// 函数类型
- (xx:number)=>number
// 类型别名
- type xx = xx
// 类型断言
- xx as string
- <string>xx
// 命名空间
- export namespace _name{ export }
- _name._xx
~~~
