# CSS适配

#### Postcss-px-to-viewport转化插件

```js
// postcss.config.js配置
plugins > "Postcss-px-to-viewport"
// 基本
- viewportWidth:375|667
- viewportUnit:'vw'
- unitPrecision:5
- minPixelValue:1
// 忽略配置
- exclude:[/tabbar/]
- selectorBlackList:['ignore']
- mediaQuery:false
/* px-to-viewport-ignore-next */
/* px-to-viewport-ignore */
```
