## Nginx

**window**:

1. conf=>nginx.conf

**Linux**-centos:

1. yum intall nginx

2. systemctl start nginx.service

   systemctl enable nginx.service

```json
// 基于location配置
server {
        listen 80;
        server_name a.fly.com;
        
    	location / { 
                root /data/web-a/dist;
                index index.html;
        }
        location /web-b { 
                alias /data/web-b/dist;
                index index.html;
        }
}
```

**特点**

~~~shell
# 高性能HTTP和反向代理web服务器
- 内存少
- 并发能力强(50000个并发响应)
# 反向代理
# 负载均衡
~~~
