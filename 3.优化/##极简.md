## For循环

```js
// for循环
for(let i,xx; xx=xxs[i++];)
for(const i in arr|obj)
for(const i of arr)
- arr.forEach((item,index,arr)=>{})
- arr.map((item,index,arr)=>{})

// while循环
- while(false){}
- do{}while(false)
```

```shell
# 内部核心
- label:for() => break|continue
```

```shell
# for in
- 遍历prototype|obj
- 可以break
# for of
- 遍历arr|类数组|map|set|str
- 可以break
# forEach和map
1.forEach|map锁定数组。map返回新数组
2.无法break
```

```js
// 类型判断
```
