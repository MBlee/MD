# VueCli

- **VueCLI4.x|3.x**

~~~shell
cnpm install -g @vue/cli
cnpm install -g @vue/cli-service-global
# 初始化项目
vue create|ui xxxPro
# 快速原型开发
vue serve MyComponent.vue
vue build MyComponent.vue
# CLI服务
vue-cli-service --open
~~~

~~~js
// vue.config.js
module.exports = {
    
}
~~~

- **VueCLI2.x**

~~~shell
cnpm install -g @vue/cli-init
# 初始化项目
vue init webpack xxxPro
~~~

~~~js
// webpack.config.js
~~~
