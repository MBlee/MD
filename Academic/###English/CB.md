#### ^For

```shell
# 目标(目的地/对象/时间/原因)
- ...for + 代词(对/为谁)(you)
- ...for + 名词/动名词(a walk)
- ...for + (完成)时间(a hour)
```

#### ^Go

```shell
# 去/到
- go home/abroad/downtown
- go to ...addr
# 某事
- go v-ing
```

#### ^Here

```shell
# 在/到这
- come here
# 这(泛指)
- Here's
```

#### ^It

```shell
# 代指物/事
- ...it
- it's...
```

#### ^This

```shell
# 这个(名词)
- this's...
- this n is...
```

#### ^To

```shell
# 介词
- ...to + 时间/地点/方向
- ...to + 间接宾语
```

```shell
# 不定式
- ...to + 目的/意图(为了...)
- 形容词 + to + 目的/原因
	want/hope/wish/expect/decide/plan
- It's ... to...
```
