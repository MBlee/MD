项目一：智慧餐厅 AI 运营中枢 (B端)
●	背景： 解决餐饮商家在多平台评价回覆难、物料补货全凭经验的痛点。
●	技术栈： Vue 3, TypeScript, NestJS, LangChain, ECharts, WebSocket
●	AI 应用：
○	智能评论回复： 接入 GPT-4/国产大模型接口，开发** AI 自动回评插件**，根据顾客评分和关键词自动生成差异化回复，回复率从 20% 提升至 95%。
○	销量预测模型可视化： 协同后端算法，将预测销量与库存预警集成到前端看板，通过 Canvas/SVG 动态渲染 模拟高峰期人流走势。
●	成果： 帮助 200+ 连锁门店降低了 15% 的食材报废率。

项目二：新一代 AI 语音点餐 H5/小程序 (C端)
●	背景： 针对高峰期人工点餐压力大、长辈使用智能设备难的问题。
●	技术栈： React, Next.js, Web Speech API, Tailwind CSS
●	核心攻关：
○	语音交互优化： 封装 Web Audio API 实现环境噪音过滤，通过前端 NLP 预处理（模糊匹配菜单关键字），将语音转文字的指令识别率在嘈杂环境下提升了 30%。
○	秒级首屏： 针对餐饮场景弱网环境，采用 SSR (服务端渲染) + 边缘计算 (Edge Runtime)，使扫码点餐页面在 2G 线下也能 1s 内呈现菜单。
●	成果： 活动期间单日支撑百万级流量，AI 辅助下单占比提升至 25%。
